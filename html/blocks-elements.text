Blocks Elements
===============

div  <-->  (divider) defines the HTML shown between each row in a repeating section
-----------------------------------------------------------------------------------------
header  <-->  used to define the header for a document or a section as it contains the information related to the title and heading of the related content
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
article  <-->  a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
address  <-->  defines the contact information for the author/owner of a document or an article
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
footer  <-->  represents a footer for its nearest sectioning content or sectioning root element. typically contains information about the author of the section, copyright data or links to related documents
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
section  <-->  represents a generic standalone section of a document
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
aside  <-->  aside (more often navbar) a portion of a document whose content is only indirectly related to the document's main content
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
main  <-->  represents the dominant content of the <body> of a document
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
nav  <-->  navigation links
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
figure  <-->  self-contained content
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
figcaption  <-->  (The Figure Caption element) represents a caption or legend describing the rest of the contents of its parent <figure> element
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
ul  <-->  unordered list of items
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
ol  <-->  an ordered list of items
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
table  <-->  An HTML structure for creating rows and columns on a Web page.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
tfoot  <-->  (The Table Foot element)
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
blockquote  <-->  specifies a section that is quoted from another source
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
pre  <-->  preformatted text
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
dd  <-->  (The Description Details element) definition, or value for the preceding term
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
dt  <-->  (Description Term element) specifies a term in a description or definition list
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
dl  <-->  (Description List element) The element encloses a list of groups of terms
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
fieldset  <-->  (The Field Set element) used to generate a caption for the <fieldset> tag when grouping related items in an HTML form
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
form  <-->  a section of a document containing normal content, markup, special elements called controls (checkboxes, radio buttons, menus, etc.), and labels on those controls
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
p  <-->  (Paragraph element) represents a paragraph
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
video  <-->  embeds a media player which supports video playback into the document
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
h1>-<h6  <-->  (Heading tags) They separate subsections of your content, which is particularly important if you're writing longer articles
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
